# Methodology

## 🏁 General Workflow
1. **Recon:** Identify technologies and endpoints.
2. **Analysis:** Understand input-output flow.
3. **Exploitation:** Develop and refine payloads.
4. **Documentation:** Record steps immediately.

## 🌐 Web Exploitation
- Check `robots.txt` and `.git` folders.
- Inspect JS files for API keys or hidden endpoints.
- Test for IDOR and Parameter Tampering.

## 📁 Forensics
- Use `file` and `strings` first.
- Check metadata with `ExifTool`.
- Deep scan with `binwalk`.
